📚 ERP 시스템 사용 가이드
📝 전표를 수정/삭제하는 방법
작성된 전표를 수정하거나 삭제하는 방법을 안내합니다. 전표 조회 화면에서 해당 전표를 선택한 후 수정 버튼을 클릭하여 내용을 변경하거나, 삭제 버튼을 클릭하여 제거할 수 있습니다.
↩️ 반품 처리하는 방법
판매한 상품의 반품 처리 절차를 설명합니다. 반품 전표를 작성하고, 재고를 조정하며, 관련 회계 처리를 완료하는 전체 프로세스를 안내합니다.
🔄 전표 항목 초기화 방법
전표 작성 화면의 입력 항목을 초기 상태로 되돌리는 방법입니다. 잘못 입력한 데이터를 한 번에 초기화하여 처음부터 다시 작성할 수 있습니다.
💰 미수금 및 미지급금 조회 방법
거래처별 미수금과 미지급금 현황을 조회하는 방법을 안내합니다. 기간별, 거래처별로 상세한 미수/미지급 내역을 확인하고 관리할 수 있습니다.
📄 명세서 위치 초기화 방법
명세서 출력 시 인쇄 위치가 틀어진 경우, 위치를 초기화하여 올바른 위치에 출력되도록 설정하는 방법을 설명합니다.
🏢 세사업장 구성(초기)방법
여러 사업장을 운영하는 경우, 각 사업장별로 독립적인 관리 체계를 구성하는 초기 설정 방법을 안내합니다.
💾 자료의 백업과 복원방법
중요한 데이터를 안전하게 백업하고, 필요시 복원하는 방법을 상세히 설명합니다. 정기적인 백업으로 데이터 손실을 예방할 수 있습니다.
💻 피시 변경(데이터 이전) 방법
컴퓨터를 교체하거나 새로운 PC로 이전할 때, 기존 데이터를 안전하게 이전하는 방법을 단계별로 안내합니다.
